== Description ==
Adds Open Graph meta tags to your Wordpress website and Buddypress Activity pages. Adds URL tag with redirected Buddypress Activity URL and Description tag. It helps Facebook Sharer to work correctly.

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `Social Graph Protocol Buddypress YR` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Click Social Graph admin menu
6. Check the options you want

= Manual Installation =

1. Upload `Social Graph Protocol Buddypress YR` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Click Social Graph admin menu
6. Check the options you want

== Frequently Asked Questions ==

= This plugin is a modification of the plugin Social Graph Protocol? =

It is a modified "Social Graph Protocol" Plugin. Adds Open Graph meta tags to your Wordpress website and into Buddypress Activity pages. Adds URL tag with redirected Buddypress Activity URL and Description tag, that enables Facebook Sharer algoritm do right. It is recommended to use with Facebook Share plugins for Buddypress.

About the Source Plugin:
-Plugin Name: Social Graph Protocol
-Description: Automatically adds Open Graph meta tags to your Wordpress website that enables your content to be shown on a users news feed when someone likes or shares your content. Also enables you to view social statistics on your content via Facebook Insights.
